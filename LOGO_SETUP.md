# Logo Setup Instructions

## Required Images

Please save the following images to complete the logo setup:

### 1. Main Logo (Institute of Herbal Medicine)
- **File name:** `logo.png`
- **Location:** `frontend/public/assets/logo.png`
- **Description:** The Institute of Herbal Medicine logo with green and gold colors
- **Recommended size:** At least 500x500px (transparent background preferred)

### 2. Partner Logo (University of the Philippines)
- **File name:** `up-logo.png`
- **Location:** `frontend/public/assets/up-logo.png`
- **Description:** The University of the Philippines official seal
- **Recommended size:** At least 400x400px (transparent background preferred)

## How to Add the Logos

1. Create the assets folder if it doesn't exist:
   ```
   frontend/public/assets/
   ```

2. Save the first image (Institute of Herbal Medicine logo) as:
   ```
   frontend/public/assets/logo.png
   ```

3. Save the second image (UP logo) as:
   ```
   frontend/public/assets/up-logo.png
   ```

4. Refresh your browser - the logos should appear automatically!
